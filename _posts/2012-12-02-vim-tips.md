+ back to last cursor position
    
        ctrl-o

+ format the code

        gg=G

+ apply vim cmd inside a file

        /* vim: set XXX=XXX: */

+ fold

        :set foldmethod=XXX
    methods(XXX) usually use 'indent' or 'marker'
    
    cmd:
    + zc fold
    + zC fold all
    + zo unfold
    + zO unfold all
    + [z goto current begin
    + ]z goto current end
    + zj goto next fold
    + zk goto last fold

+ undo

        u

+ redo

        ctrl-r
